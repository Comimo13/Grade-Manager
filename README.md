# Grade Manager 📊

Grade Manager is a simple Python project that allows you to track and manage your school grades by subject.

## 💡 Features

- Add grades for different subjects
- Remove grades
- Calculate average for one subject
- Calculate overall average
- View all grades
- Automatic saving to JSON file

## 🧠 How it works

Grades are stored in a dictionary where each subject has its own list of grades.  
All data is saved in a JSON file (`grades_v3.json`), so progress is not lost after closing the program.

## 📚 Subjects included

- Math
- Physics
- Chemistry
- Biology
- Uzbek language
- Russian language
- English
- History
- Geography
- Informatics
- Art
- Physical education

## 💾 Data Storage

- File: `grades_v3.json`
- Automatically loads on startup
- Saves after changes
